# Packaging

This document describes the **product lines Yeoul may expose**.
It does not imply equal priority for all lines in the current repository phase.

## Current priority

The active priority order is:

1. Yeoul Core
2. Yeoul CLI
3. Yeoul Agent Pack

The daemon and other adapters remain optional and deferred until the embedded toolkit is proven.

## Yeoul Core

Go library. No AI dependency.

## Yeoul CLI

Developer tool for local inspection, migration, ingest, and search.

## Yeoul Daemon

Optional local service for multi-process access.

## Yeoul Agent Pack

Skills, instructions, ontology templates, and search recipes for AI agents.

## Yeoul Examples

Reference applications showing how to use Yeoul with coding agents, research agents, and local assistants.
