## How these references are used
Yeoul is intentionally **not** a Graphiti port. However, Graphiti informed the scope of:
- episode-centric memory
- temporal facts
- provenance requirements
- retrieval-oriented graph design

Ladybug informed the scope of:
- embedded local-first storage
- schema-first property graph design
- Cypher usage
- persistence expectations
- concurrency restrictions
- Go integration constraints
