# Non-goals

Yeoul Core is not:

- an AI agent framework
- an LLM orchestration framework
- a prompt management system
- a vector database replacement
- a remote multi-tenant graph database server
- a general-purpose Cypher abstraction layer
- a full Graphiti reimplementation
- a Neo4j-compatible server
- a workflow engine
- a task planner
- a chat memory product by itself

Yeoul may provide optional packages for AI agents, but those packages must remain outside the core engine.
