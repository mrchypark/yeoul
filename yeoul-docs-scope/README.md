# Yeoul Docs Scope Archive

This directory used to hold a separate scope draft for Yeoul documentation.

That draft has been folded into the canonical documentation tree under [`docs/`](../docs), so the duplicate document set was removed.

Use these paths instead:

- [`docs/README.md`](../docs/README.md): documentation map
- [`docs/00-overview/current-scope.md`](../docs/00-overview/current-scope.md): current scope anchor
- [`agent-pack/`](../agent-pack): agent-facing starter policy pack

This directory remains only as a compatibility pointer for anyone who still has an old reference to `yeoul-docs-scope/`.
